disconnectReceiver

  • disconnectReceiver(obj: any): void
    • Defined in index.ts:220

    Remove all connections where the given object is the receiver.

    Parameters

    • obj: any

      The receiver object of interest.

      Notes

      If a thisArg is provided when connecting a signal, that object is considered the receiver. Otherwise, the callback is used as the receiver.

      Example

      // disconnect a regular object receiver
disconnectReceiver(myObject);

// disconnect a plain callback receiver
disconnectReceiver(myCallback);

    Returns void

signal

  • signal(proto: any, name: string): void
    • Defined in index.ts:142

    A decorator which defines a signal for an object.

    Parameters

    • proto: any

      The object prototype on which to define the signal.

    • name: string

      The name of the signal to define.

      Notes

      When defining a signal from plain JS (where decorators may not be supported), this function can be invoked directly with the class prototype and signal name.

      Example

      class SomeClass {

  @signal
  valueChanged: ISignal<number>;

}

// ES5 alternative signal definition
signal(SomeClass.prototype, 'valueChanged');

    Returns void
